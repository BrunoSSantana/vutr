# VUTR

## About
VUTR is a simple API created for the purpose of learning how to create a RESTful API technology how to graphql and how to use it in a project.

## Installation
1. Clone the repository
2. Install the dependencies
```bash
npm install
```
3. Run the server
```bash
npm run start:dev
# or
npm run start
```
